Option A Most flexible

Equal Grid of Cards

All pieces are the same size in a 3-column grid. Clean, symmetric, and very easy to add/remove pieces. Each card shows media (image or video thumbnail), a title, and a short description. Dashed placeholders show where to add new work.

Option C

Horizontal List / Timeline

Each piece is a full-width horizontal row: thumbnail on the left, title + description + tag on the right with a colored accent border. Very readable and detailed — great for clients who want to read about the work. Feels more like a case study list.
